Servidor exclusivo para Ridder Productive

Situación actual vs beneficios esperados con servidor exclusivo y licencia de Windows y SQL.

Actual

  • !Servidor compartido con Sys21 y otros programas/servicios del predio.
  • !Recurso compartido: las políticas de seguridad interfieren actualmente con la conexión directa desde equipos de Vhggies.
  • !No se tiene una copia de seguridad automática.
  • !SQL Express limitado a 10 GB en la base de datos, con riesgo de bloqueo de Ridder.
  • !Windows Server 2016 sin licencia activa.

Con servidor exclusivo

  • ✓Menor probabilidad de que servicios y programas interfieran en la funcionalidad de Ridder.
  • ✓NS podría agregar conexión directa al servidor sin interferir con programas o servicios actuales del predio.
  • ✓Copias de seguridad automáticas diarias en un servicio independiente para restaurar el sistema en otro servidor si fuera necesario.
  • ✓SQL completo: sin límite en la base de datos y sin riesgos de bloqueo o pérdida de información.
  • ✓Windows Server 2019 con soporte del proveedor y compatibilidad completa con apps de Ridder en versión de escritorio.
